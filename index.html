Bootcamp 2020: Infortmation about cities in the UK

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of if/else statements. I created a map of the Uk through using images in proccessing and finding each of the cities coordinates by printing x and y. With the help of if/else statements the map allows you to hover around the cities i have selected and display the information about these cities. Key: - London - Edinburgh - Belfast - Cardiff - Liverpool - Manchester
